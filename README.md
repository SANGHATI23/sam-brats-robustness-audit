# SAM Medical Robustness Audit

A structured robustness evaluation of the Segment Anything Model (SAM) in medical imaging using controlled perturbation experiments.

This repository contains a fully reproducible pipeline to evaluate foundational segmentation models under domain shift conditions in medical datasets.

---

## 1. Study Objective

This project evaluates the robustness of the Segment Anything Model (SAM, ViT-B) for medical image segmentation under controlled domain perturbations.

We measure:

- Baseline segmentation performance
- Sensitivity to noise
- Sensitivity to contrast shifts
- Sensitivity to spatial resolution changes
- Sensitivity to bias field intensity drift

The goal is to assess whether a foundational vision model maintains stability under realistic imaging perturbations.

## 2. Dataset Source

### Brain Tumor Dataset (BraTS-derived)

- Multi-modal MRI brain tumor dataset
- Public medical segmentation benchmark
- 3D volumes converted to 2D slices for evaluation
- Only slices containing tumor regions are used

## 3. Model Details

Segment Anything Model (SAM)

- Architecture: ViT-B
- Checkpoint: `sam_vit_b_01ec64.pth`
- Source: https://github.com/facebookresearch/segment-anything
- No fine-tuning performed
- Box-prompt based inference

## 5. How to Run

1. Open the notebook in Google Colab.
2. Enable GPU.
3. Run all cells sequentially from top to bottom.
4. The notebook will:
   - Install SAM
   - Load dataset
   - Generate bounding box prompts
   - Run segmentation
   - Apply perturbations
   - Compute evaluation metrics
   - Save results

---

## 6. Perturbations Included

Four domain shift categories:

### 1. Gaussian Noise
- σ = 5
- σ = 15
- σ = 30

### 2. Contrast Scaling
- α = 0.9
- α = 0.7
- α = 0.5

### 3. Resolution Downsampling
- 0.75×
- 0.5×
- 0.25×

### 4. Bias Field Drift
- Strength 0.1
- Strength 0.2
- Strength 0.35

Each perturbation is applied independently.

---

## 7. Metrics Computed

For each slice:

- Dice coefficient
- Intersection-over-Union (IoU)
- Sensitivity
- Specificity
- SAM confidence score

Additional analysis:

- Relative Dice drop (%)
- Failure rate (Dice < 0.5)
- Wilcoxon paired statistical test
